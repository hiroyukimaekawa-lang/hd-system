# Google Maps Lead Scraper - Keyword Mode

Googleマップから営業リストを取得するChrome拡張機能です。
固定ジャンル一覧ではなく、任意の「エリア」「検索キーワード」「CSV上のジャンル」を入力して取得します。

## 主な機能

- 任意キーワード検索
- Googleマップ検索結果のオートスクロール
- 検索結果URLの収集
- 施設/店舗詳細を一件ずつクリックして取得
- 店名、住所、電話番号、公式HP、営業時間、GoogleマップURLを取得
- CSV / XLS出力
- STOP時も取得済みデータを保存
- 一括取得対応

## 使い方

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. この `google-maps-scraper` フォルダを選択
5. 拡張機能を開く
6. 以下を入力
   - エリア: `千葉県 館山市`
   - 検索キーワード: `グランピング`
   - CSV上のジャンル: `グランピング`
7. 「取得開始」を押す

## 入力例

### グランピング

- エリア: `千葉県 館山市`
- 検索キーワード: `グランピング`
- CSV上のジャンル: `グランピング`

### コテージをグランピング扱いで取得

- エリア: `千葉県 南房総市`
- 検索キーワード: `コテージ`
- CSV上のジャンル: `グランピング`

### アパレル個人店

- エリア: `千葉県 千葉市`
- 検索キーワード: `古着屋`
- CSV上のジャンル: `アパレル`

### 中古車関連

- エリア: `千葉県 木更津市`
- 検索キーワード: `中古車販売`
- CSV上のジャンル: `中古車関連`

## 一括取得

一括取得欄に以下の形式で入力できます。

```text
エリア, 検索キーワード, CSV上のジャンル
```

例:

```text
千葉県 館山市, グランピング, グランピング
千葉県 南房総市, コテージ, グランピング
千葉県 富津市, 貸別荘, グランピング
千葉県 千葉市, 古着屋, アパレル
千葉県 木更津市, 中古車販売, 中古車関連
```

## CSV出力

基本列:

```csv
店名,ジャンル,検索ジャンル,取得元ジャンル,都道府県,市区町村,住所,電話番号,定休日,営業日,営業開始A,営業終了A,営業開始B,営業終了B,営業時間原文,URL,HP有無,媒体,取得元URL,取得日時
```

- ジャンル: ユーザーが入力した「CSV上のジャンル」
- 検索ジャンル: Googleマップ検索に使ったキーワード
- 取得元ジャンル: Googleマップ上で取得できたジャンル

## 注意点

- Googleマップの画面構造が変わると、`content.js` のセレクタ調整が必要になる場合があります。
- スクレイピング側では強い除外判定をしません。チェーン除外、ビル管理判定、商業施設判定、重複排除はGAS側で行ってください。
- グランピング/宿泊系取得で必要なため、ホテル・キャンプ場などの飲食店向け除外フィルタは任意キーワードモードでは無効化しています。
