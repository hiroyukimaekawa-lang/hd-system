# Gourmet List Scraper (食べログ・ホットペッパー 飲食店リスト取得ツール)

主要なグルメサイト（食べログ、ホットペッパー、Google Maps）の検索結果から、飲食店情報を自動で抽出し、CSV形式でダウンロードできるChrome拡張機能です。

## 🌟 主な機能
- **一括抽出**: 検索結果一覧ページから店舗名、電話番号、住所などの情報を自動取得。
- **マルチサイト対応**: 食べログ、ホットペッパー、Google Mapsの各ドメインで動作します。
- **CSVダウンロード**: 取得したデータをそのままCSV形式で保存可能。
- **自動通知**: 抽出完了時にデスクトップ通知でお知らせ。

## 🛠 対応サイト
- [食べログ (tabelog.com)](https://tabelog.com/)
- [ホットペッパーグルメ (hotpepper.jp)](https://www.hotpepper.jp/)
- [Google Maps (google.com/maps)](https://www.google.com/maps/)

## 🚀 インストール方法
1. このリポジトリを右上の `Code` > `Download ZIP` からダウンロード、または `git clone` します。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` を入力して拡張機能画面を開きます。
3. 右上の「デベロッパーモード」を **ON** にします。
4. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
5. ダウンロード（展開）したプロジェクトフォルダを選択すると、拡張機能がインストールされます。

## 💡 使い方
1. 対応サイト（例：食べログの検索結果ページ）を開きます。
2. 拡張機能のアイコンをクリックしてポップアップを開きます。
3. 「リスト取得開始」ボタンを押すと、現在のページの店舗情報が抽出されます。
4. 抽出完了後、自動的にCSVファイルがダウンロードされます。
