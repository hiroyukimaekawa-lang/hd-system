# CSV統合・正規化・重複排除・チェーン店除外・コムデスクCSV生成 GAS

本ディレクトリには、HD事業部の営業リスト作成業務でスプレッドシート上に実装・運用しているGoogle Apps Script (GAS) の機能概要と、コードテンプレートを配置します。

## 主な機能

### 1. CSV統合・正規化機能 (`csv-normalize-gas`)
- Googleマップ、食べログ、ホットペッパーから取得したCSVデータをスプレッドシートに一括取込
- 文字コード（UTF-8, Shift_JIS）を自動判定して文字化け防止
- 店名の表記揺れ補正、電話番号のハイフン正規化、住所からの都道府県・市区町村の自動分割抽出

### 2. 重複排除機能 (`dedup-gas`)
- 複数媒体で重複している店舗を以下のルールに基づき自動判定・統合
  - 電話番号が一致：重複確定
  - 店名＋住所が一致：重複確定
  - 店名のみ一致：重複候補（「要確認」フラグを付与）
- 重複時は「Googleマップ」から取得した情報を最優先し、不足している項目を食べログ・ホットペッパー情報から補完します。

### 3. チェーン店除外機能 (`chain-exclude-gas`)
- リスト作成の基準リストである `MASTER_CHAIN` シートと突合し、大手チェーン店を自動除外
- 店名に含まれるキーワード（「チェーン」「グループ」等）の自動判定
- 同一店名が大量（例: 5店舗以上）出現した場合にチェーン候補として「チェーン候補」タブへ抽出

### 4. 電話番号補完機能 (`phone-supplement-gas`)
- 電話番号が空白の店舗情報を抽出し、外部APIを呼び出して電話番号を自動補完
- **重要**: APIキーはソースコード内に直接書き込まず、必ずスプレッドシートの **「スクリプトプロパティ」** (`PropertiesService.getScriptProperties()`) から取得するように設計します。

### 5. コムデスク投入用CSV生成機能 (`comdesk-csv-gas`)
- コムデスクのインポート形式に合わせたヘッダーに自動整形
- `都道府県_市区町村_業種.csv`（例: `栃木県_宇都宮市_居酒屋.csv`）の単位で自動分割・ダウンロード
